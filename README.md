## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Dart
```sh
brew install dart
```

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/dangilbert/monica.git
```
2. Install Correct flutter version (Currently using 1.12.13+hotfix.5)
```sh
fvm install stable
fvm use stable
```
3. Run the project
```sh
./fvm run
```
